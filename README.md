# Retriever
!<img src="favicon.png" alt="Retrieverアイコン"width="200" height="200">

Docker上で動作する宅内LAN用Webアプリケーションです。
画像ファイルや動画ファイルを収集し、利用しやすいように取り出せるよう管理します。
